# החווה לחינוך חקלאי באר שבע — Landing Page

Landing page for **The Agricultural Education Farm Beer Sheva** ("יקב החווה" / Farm Winery Since 2015), run by Dr. Ilan Avitbol.

The farm sells handmade products — wine, natural skincare, mosquito repellent, massage oils, herb plants, herbal teas, and curated gift packages — primarily to schools and education teams.

## Stack

- Single `index.html` (inline CSS + JS)
- Hebrew RTL
- Mobile-first
- Accessibility panel (text size, contrast, reduced motion)
- No build step, no external images — emoji + CSS only

## Local preview

```bash
python -m http.server 8771
```

Then open <http://127.0.0.1:8771>.

## Deployment

Static — drop into Netlify, Vercel, or any static host.
